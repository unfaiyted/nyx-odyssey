# Odyssey

TanStack Start + React application running on port 3003.

## Tech Stack

- **Framework:** TanStack Start + React
- **Styling:** Tailwind CSS v4
- **Database:** PostgreSQL + Drizzle ORM
- **Charts:** Recharts
- **Animation:** Framer Motion

## Development

```bash
bun install
bun dev
```

## Database

```bash
bun db:generate
bun db:push
```
